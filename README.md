# Tauri 2 Svelte 5 Boilerplate

Simple boilerplate for Tauri 2 with Svelte 5 (and DaisyUI) using Bun runtime.

## Useful commands
### Start dev server
```

bun run tauri dev

```

### Build executable
```

bun run tauri build

```
